String Dan Text

Pada belajar kali ini masih tentang string, karena sebagain orang masih bingung dengan string pada awalnya termasuk saya :) tapi setelah pada bab ini saya sedikit paham tentang string itu. String sebenarnya adalah sebuah teks yang ingin kita tampilkan pada program. Python dapat mengenali sebuah string dengan adanya tanda “ (petik dua) atau ‘ (petik satu). String dapat berisikan banyak variabel pada script Python. Untuk diingat bahwa variabel adalah setiap kode yang ada nama variable dan nilai, sebagai contoh saya = manusia, yang mana saya adalah nama dari variabel dan manusia adalah nilai dari nama variabel.
