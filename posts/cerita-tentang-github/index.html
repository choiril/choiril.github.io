Penggunaan Github dan Git Pemula

January 5, 2017

Reading time ~2 minutes

Sebuah cerita nyata mengenal dan menggunakan Github dan Git dari seorang pemula. Awal penasaran tentang teknologi git yang begitu menggoda para developer dan cara instalasi git pada PCLOS.

Berkenalan dengan Github

Apa itu Github?Github sebenarnya hampir mirip dengan sosial media hanya saja berisi banyak repositori sumberkode yang dikerjakan dan atau ditulis bersama untuk pengembangan perangkat lunak yang menggunakan sistem pengontrol versi git sehingga juga disebut social coding. Di github ini kita bisa punya pengikut dan mengikuti orang,memberikan umpan, wiki, grafik pengembangan, dan berkonstribusi dalam pembuatan sebuah perangkat lunak selain itu pula kita juga bisa membuat web statis dengan github pages. Pertama mengenal github waktu masih kuliah semester 2, waktu itu ada dosen memberitahukan membangun sebuah perangkat lunak dapat dilakukan dengan banyak orang meski jarak yang jauh dan tak mengenal orang tersebut. Pada saat itu terbesitlah pertanyaan jika seandainya sumberkode kita disunting seseorang dan menanamkan sebuah kode yang bersifat jahat? karena github menggunakan git sebagai kontrol versi maka ada pemeberitahuan kalau ada perubahan sumberkode dan siapa yang telah melakukan perubahan kode tersebut. Apa yang dimaksud dengan Git?dari penjelasan diatas dapat disimpulkan bahwa git adalah perangkat lunak pengontrol versi yang diciptakan oleh Linus Torvalds, yang awalnya ditujukan untuk pengmebangan kernel Linux.

Mencoba Github

Saat kuliah mencoba Github dengan Windows alasan tersebut dapat ditemukan dalam postingan saya sebelumnya. Mulailah menginstal Git GUI di Windows dan mendaftar di Github dan membuat sebuah github pages dan repository kosong tentang web, selain itu mulai mencari tutorial tentang Github namun setelah beberapa bulan minat menggunakan Github menurun karena kurangnya fokus dan bingung mau diisi dengan apa.

Lebih dekat dengan Github

Saat ini saya mencoba untuk mengenal lebih dekat Github mengunakan PCLinuxOS lebih tepatnya memanfaatkan fitur githubpages, untuk memulai itu dengan menginstal paket Git: apt-get install gitdan Jekyll: gem install jekyll. Pada Desember kemarin satu bulan sudah mencoba belajar dari awal dan menggunakan Github, Jekyll dan Git. Tepat 31 Desember weblog ini berhasil diupload ke Github, kenapa saya mengatakan berhasil? karena saya masih bingung dengan perintah Git waktu itu seperti mengupload berkas ke Github. Weblog ini menggunakan tema Minimal dari Blacktie dengan beberapa penyesuaian yang saya bisa. Awalnya mau menggunakan tema Munter Design tapi karena kendala kostumisasi yang sedikit rumit bagi saya sehingga menggunakan Minimal Themes. Ada sedikit cerita lain tentang Github ini, teman lama SMA saya juga tertarik menggunakan Github untuk membuat web statis setelah saya tunjukkan beberapa web yang menggunakan Jekyll sebagai platformnya. Kata dia terlihat cepat loadingnya dan desainnya responsiv.
