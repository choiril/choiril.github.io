Challenges Zero Days

January 28, 2018

Reading time ~2 minutes

Challenges Zero Days? apa itu? sesuai dengan artinya inilah hari ke-0 tantangan dari tantangan 100 hari belajar koding, sebuah gerakan yang cukup terkenal didunia dengan hastag 100DaysofCode pada sosial media terutama Twitter. 100DaysofCode adalah bagian dari gerakan 100DaysofX, 100DaysofX adalah suatu gerakan selama seratus hari untuk mempelajari sesuatu secara konsisten, apakah kamu tertarik mengikuti tantangan ini?

Tantangan Kepada Diri Sendiri

Kali ini saya akan menantang diri saya sendiri untuk 100dayofcode dan 100dayofwriting, hari ini adalah hari ke-0 saya untuk memulai. Kenapa ke-0 bukan langsung ke-1? karena saya mau memantapkan niat dulu untuk menulis 100 hari secara konsisten dan belajar pemrograman Python 100 hari secara konsisten. Pada awalnya saya bingung untuk memulai 100dayofcode diawali darimana, pertama saya menemukan Pybtes yang merupakan sebuah website untuk belajar pemrograman Python yang ada tantangan selama 100 hari dan tantangan yang menurut saya cukup wah. Saya sendiri sudah fork repository mereka, lihat beberapa kode saya masih belum paham dan pada akhirnya saya menemukan ebook Learn Python 3 The Hardway. Learn Python 3 The Hardway adalah salah satu ebook yang di rekomendasikan untuk pertama kali belajar bahsa pemrograman Python, dalam ebook ini juga tersedia contoh dan latihan kode dan penjelasan kode yang ditulis.

Kenapa Memilih Pemrograman Python?

Saya tertarik mempelajari Python karena saya ingin membuat sebuah chatbot yang dibekali kecerdasan buatan. Selain itu saya saat ini bekerja sebagai data entri, entah suatu saat nanti data yang saya entri pasti perlu sebuah analisa, saya ingin menggunakan Python untuk menganalisa data yang saya entri. Dan Python juga dianjurkan untuk pemula yang ingin mendalami pemrograman, Python juga bisa untuk pembuatan webapp, aplikasi desktop yang crossplatform, aplikasi smartphone, kecerdasan buatan dan masih banyak lagi.

Kenapa 100DaysofWriting?

Untuk alasan ini sebagai bentuk dokumentasi dari 100DaysofCode, yang mana setiap kode dan penjelasannya akan saya tulis di weblog ini. Kecuali hari sabtu dan minggu yang Insya Allah akan saya terbitkan pada hari senin. Walau terbit hari senin tapi hari sabtu dan minggu akan saya tulis dalam bentuk draft postingan dan saya tetap menjalankan 100DaysofCode. Hal ini dikarenakan koneksi internet di rumah kurang memadai, tapi jika ada koneksi akan saya langsung terbitkan pada hari tersebut. Jika hari senin saya melakukan 100DaysofCode maka pada hari selasa saya akan posting tentang hasil belajar saya pada hari senin kemarin. Untuk 100daysofCode akan saya posting pada media sosial facebook saya.

Mari Bergabung!

Bergabunglah, untuk menantang diri sendiri dan jadilah yang terbaik untuk dirimu.
