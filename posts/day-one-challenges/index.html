Day One Challenges

January 29, 2018

Reading time ~2 minutes

Hari ini adalah hari pertama menghadapi tantangan melakukan 100 hari belajar Python dan menulis. Silahkan baca tulisan ini, dan jika tertarik mari kita belajar bersama dan terima tantangan #100dayofcode dan #100dayofwriting.

Tantangan Belajar Hari Pertama

Pertama saya membuat folder baru dengan nama 100daysofcode, folder ini akan berisikan file latihan yang saya tulis menurut panduan ebook Learn Python 3 The Hard Way, karena saya pengguna Linux maka menggunakan perintah mkdir 100daysofcode. Perintah ini berfungsi untuk membuat folder yang bernama 100daysofcode, setelah itu buka text editor kesukaan kalian disini saya menggunakan SublimeText3.

Menulis Baris Kode Sederhana

Pada belajar hari pertama ini kita akan belajar print, sesuai buku tersebut kita akan menulis baris kode berikut:

print("hello world!")
print("Hello again")
print("I like typing this")
print("This is fun.")
print('Yay! Printing.')
print("I'd much rather you 'not'.")
print('I "said" do not touch this.')

Baris kode diatas tidaklah boleh kita menyalin dari bukunya, kita harus menulisnya sendiri. Dengan menulis sendiri kita bisa tahu maksud dari baris kodenya lalu simpan dengan nama lat1.py di folder 100daysofcode, jalankan file tersebut dengan perintah

python lat1.py

Maka akan menghasilkan keluaran berikut :

Untuk exercise pertama kita disuruh untuk menulis sendiri kode kita dan menambah tanda #, disini saya membuat 2 kode baris baru

#print("Hasil latihan dengan menambah tanda #")
print("Hasil latihan Python 3 menggunakan print")

Jalankan perintah seperti sebelumnya

Dari hasil tersebut dapat disimpulkan ketika kita menambah tanda # maka baris kodenya tidak akan tampil.Untuk hari pertama cukup sampai disini, kelihatannya mudah tapi yang sulit adalah merangkainya menjadi kata mungkin karena hari pertama jadi sedikit berantakan. Terima kasih
