For Loop Python

February 28, 2018

Reading time ~2 minutes

Kalau ada percabangan pasti ada perulangan, kali ini pada hari ke-28 kita akan belajar tentang perulangan. Untuk hari ini perulangan menggunakan for-loop, tapi sebelum kita menggunakan perulangan kita haru menyimpan terlebih dulu apa yang akan kita ulangi. Untuk menyimpan kita butuh sesuatu yaitu list atau daftar, contoh list:

angka = [1, 2, 3, 4, 5]
hewan = [kelinci, marmut, hamster]

Lebih mudahnya, langsung saja ke kode yang sudah saya buat sesuai study drill:

# membuat list
angka = [1, 2, 3, 4, 5]
buah = ['apel', 'mangga', 'anggur', 'jeruk', 'durian']
ang_ker = [1, 'siji', 2, 'loro', 3, 'telu']

# membuat for-loop di list
for x in angka:
	# akan mencetak sebanyak list yang ada
	print(f"Ini adalah angka ke-{x}")

# membuat for-loop di list
for y in buah:
	# akan mencetak sebanyak list yang ada
	print(f"Ada buah {y}")

# for-loop di list dapat digabungkan antara angka dan karakter
for z in ang_ker:
	# akan mencetak sebanyak list yang ada
	print(f"Angka dalam bahasa jawa: {z}")

# kita dapat membuat list kosong 
daftar = []

# menggunakan fungsi range, dengan range(0,6) akan membuat list dari angka 0 sampai 5 
for a in range(0, 6):
	print(f"Menambahkan angka {a}")

# dengan fungsi append kita dapat mendapatkan hasil list yang tidak dapat kita ketahui tapi biasanaya adalah list terakhir
daftar.append(a)

# membuat for-loop di list daftar
for a in daftar:
	# akan mencetak sebanyak list yang ada
	print(f"Ini adalah: {a}")
