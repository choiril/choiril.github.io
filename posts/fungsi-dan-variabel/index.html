Fungsi Dan Variabel

February 18, 2018

Reading time ~2 minutes

Sebelum ke intinya saya akan sedikit bercerita tentang pelajaran tambahan yang saya pelajari, yaitu belajar scrape data menggunkan Python. Meski masih awam, tapi skripnya masih menggunakan fungsi jadi sedikit lebih paham. Saya belajar scrape untuk menunjang pencarian data, tapi masih belum terlalu jauh baru mencari data seadanya. Bahkan ada web yang belum bisa saya scrape padahal cuma mencari jumlah dataset pada https://data.go.id/dataset yang menggunakan tag html saja, output yang saya dapat hanya kosong tidak ada angkanya. Beda dengan https://data.gov masih bisa menghasilkan output.

Hari ke-19 masih tentang fungsi, fungsi dapat digabungkan dengan variabel. Dan masih sedikit berkait dengan pelajran tambhan yang saya ambil hari ini. Berikut kode yang saya ketik ulang dan memberikan komentar.

# membuat fungsi pisang_dan_keju
def pisang_dan_keju(jml_pisang, jml_keju):
	print(f"Kita punya pisang sebanyak {jml_pisang} buah.")
	print(f"Dan keju sebanyak {jml_keju} buah.")
	print("Jumlah yang cukup untuk membuat pisang keju")
	print("\n")


# memanggil fungsi secara langsung
print("Menggunakan fungsi scara langsung")
pisang_dan_keju(20, 30)


# memberikan variabel sendiri
print("Memberikan variabel dari skrip")
total_pisang = 20
total_keju = 70
# memanggil fungsi
pisang_dan_keju(total_pisang, total_keju)


# menggunakan operator matematika di fungsi
print("Menggunakan operasi matematika")
pisang_dan_keju(40 + 35, 23 + 6)


# penggunaan variabel dan matematika
print("Menggunakan variabel dan amtematika")
pisang_dan_keju(total_pisang + 5, total_keju + 30)


# penggunaan input dan fungsi
print("Menggunakan input")
prompt = "> "
print("Berapa sisa pisang")
sisa_pisang = input(prompt)
print("Berapa sisa keju")
sisa_keju = input(prompt)
pisang_dan_keju(sisa_pisang, sisa_keju)
