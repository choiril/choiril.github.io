Hari Kedua Tantangan

January 30, 2018

Reading time ~2 minutes

“Rasa malas akan terus menjalar kedalam diri, hanya dengan hati yang tangguh rasa tersebut akan kalah. Kemalasan memiliki dua mata yang cukup tangguh, jikalau digunakan dengan benar maka akan menemukan sebuah inovasi baru. Namun ketika terbalik akan menimbulkan penyesalan.”

Hari ini adalah hari kedua saya melakukan belajar pemrograman Python 3, saya akui pada hari ini saya merasa malas gara-gara tadi keasyikan bermain dengan inkscape setelah lihat-lihat karya di grup telegram gimscape indonesia :) jadi saya tertarik untuk membuat karya. Dan hasilnya masih belum sesuai dengan ide desainnya.

Pada hari ini masih lanjutan dari kemarin tentang tanda pagar (#), tanda ini sangatlah penting bagi penulis kode. tanda # ini diberikan untuk memberikan keterangan tentang kode yang ditulis, kalau bahasa programmernya disebut komentar. Untuk lebih jelasnya silahkan tulis baris kode berikut pada teks editor kalian :

# A comment, this is so you can read your program later.
# Anything after the # is ignored by python.
print("I could have code like this.") # and the comment after is ignored
# You can also use a comment to "disable" or comment out code:
# print("This won't run.")
print("This will run.")
#komentar tidak akan dibaca oleh program
#komentar pada python di awali dengan tanda pagar (#)
print("Ini akan ditampilkan dalam program")
#print("karena ada tanda #, maka tidak dibaca oleh program")

Simpan kode tersebut dan beri nama lat2.py atau bisa menyalin dari repositori saya, untuk pada bab ini pada ebook Learn Python 3 The Hardway latihannya hanya mengamati apakah ada salah atau tidak dalam kode. Karena itu saja tulisan ini juga sedikit, pasti ada pertanyaan kenapa tidak langsung 2 bab sekaligus kalau sedikit. Hal ini dikarenakan saya mencoba untuk konsisten satu hari hanya satu bab, tapi entah untuk bab yang sudah mendekati angka puluhan apaa akan selesai untuk satu hari satu bab :)

Terima kasih sudah membaca.
