Matematika Dalam Python

January 31, 2018

Reading time ~2 minutes

Hidup ini tak pernah lepas dari matematika dan perhitungan. Setiap hari kita berhitung, tapi jangan sekali-kali perhitungan dengan teman :). Tapi kalau temannya sudah kelewat batas bolehlah dikasihperhitungan.


Setiap bahasa pemrograman punya cara tersendiri untuk melakukan perhitungan matematika. Pada hari ketiga ini saya belajar untuk melakukan perhitungan matematika dan menggunakan penjumlahan, pengurangan, perkalian, pembagian, modulus, kurang dari, lebih dari, kurang dari sama dengan, lebih dari sama dengan. Berikut hasil penulisan kodenya yang mana sudah saya lengkapi dengan keterangan dan hasil dari study drill pada ebook Learn Python The Hard Way :

#perhitungan matematika pada Python 3
print("Menghitung jumlah ayam:")
#menggunakan operator matematika pertambahan dan pembagian
print("Ayam betina", 40 + 50 / 1)
#menggunakan operator pengurangan, perkalian dan modulus
print("Ayam jantan", 94 - 26 * 2 % 2)
print("Menghitung telur ayam:")
#penggabungan semua operator matematika
print(4 + 6 + 9 - 5 + 4 % 4 - 1 / 6 + 6)
print("Apakah benar 3 + 2 < 5 -7?")
#menggunakan operator kurang dari, jika benar akan mengeluarkan true dan jika salah akan menampilkan false
print(3 + 2 < 5 - 7)
#penjumlahan
print("berapa hasil dari 3 + 2?", 3 + 2)
#pengurangan
print("Berapa hasil dari - 7?", 5 - 7)
#menggunakan operator lebih dari
print("Apakah 5 > -2?", 5 > -2)
#menggunakan operator lebih dari sama dengan
print("Apakah 5 lebih besar dari sama dengan -2?", 5 >= -2)
#menggunakan operator kurang dari sama dengan
print("Apakah 5 kurang dari sama dengan -2?", 5 <= -2)
#untuk operator kurang dari dan lebih dari akan menghasilkan keluaran true atau false
#menggunakan nilai floating, floating point adalah penggunaan koma
print("berapa hasil dari 20.0 + 45.0 - 10.0 / 2.0 * 5 % 2 =", 20.0 + 45.0 - 10.0 / 2.0 * 5 % 2)
