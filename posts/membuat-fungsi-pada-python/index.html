Membuat Fungsi Pada Python

February 17, 2018

Reading time ~2 minutes

Gini nih kalau koneksi internet masih menumpang di sekolah dekat rumah, banyak yang akses dan lemotnya itu lho membuat tidak sabar walau begitu belajar tak boleh ketinggalan. Pada latihan ke-18 ini belajar membuat sebuah fungsi pada Python. Fungsi pada Python dapat dibuat dengan def, dengan fungsi kita dapat memanggil lagi kode yang telah ditulis sehingga kita tidak perlu repot-repot menulis ulang apalagai kalu kodeya panjang membuat malas menulisnya lagi jadi cukup memanggil fungsi pendek yang telah kita buat maka inti dari kodenya bisa dibuat. Apakah masih bingung?

Langsung saja ke kode yang saya buat sesuai ebooknya, untuk kode kali ini study drillnya agak membingungkan jadi saya tuils ulang dan rubah ke bahasa Inonesia yang tentunya dengan pemberian komentar setiap baris kodenya:

# belajar membuat fungsi pada Python

# membuat fungsi1 yang berisikan argeumen "kata", def kepanjangan dari define. yang artinya menentukan sebuah fungsi
# setiap pembuatan fungsi diakhiri dengan :, setelah : kode selanjutnya akan menjorok kedalam
# fungsi1 adalah nama sebuah fungsi
def fungsi1(*kata):
# ini adalah isi dari fungsi
	kata1, kata2 = kata
	print(f"Kata pertama: {kata1}, kata kedua: {kata2}")

# membuat fungsi1_lagi yang mana pada fungsi ini argumen lebih dibuat simpel
def fungsi1_lagi(kata1, kata2):
# ini adalah isi dari fungsi
	print(f"Kata pertama: {kata1}, kata kedua: {kata2}")

# membuat fungsi2 dengan satu argumen
def fungsi2(kata1):
# ini adalah isi dari fungsi
	print(f"kata pertama: {kata1}")

# membuat fungsi3 tanpa argumen
def fungsi3():
# ini adalah isi dari fungsi
	print("Tidak ada kata")

# memanggil fungsi 
fungsi1("variabel satu", "variabel dua")
fungsi1_lagi("variabel tiga", "variabel empat")
fungsi2("variabel lima")
fungsi3()
