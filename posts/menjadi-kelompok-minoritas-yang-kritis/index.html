March 11, 2017

Reading time ~3 minutes

Menjadi Kelompok Minoritas Yang Kritis.

Ini berawal saat mulai bekerja, mulai mengintip OS yang dipakai di lingkungan kerja. Dan yah, ini lingkungan kerja dengan pengguna Windows tak ada seorangpun yang menggunakan Linux hanya saya seorang. Saya merasa tidak kesulitan walau dalam lingkungan seperti ini karena saya sudah pernah untuk input data ke aplikasi berbasis web, kerjaan yang sudah pernah saya lakukan saat masih magang dulu di BPMPD untuk inuting buku profil desa ke aplikasi prodeskel berbasis web, dan menurut saya tak ada kendala sama sekali.

Tapi pemikiran itu beruah saat disodorkan sebuah file spreadsheet yang harus diupload ke web dawis, spreadsheet tersebut ada file ekstensi xls yang terdapat makro. Ketika file berektensi xls tersebut di upload keluar sebuah peringatan The filetype you are attempting to upload is not allowed , apa ini? itulah yang terlintas dalam benak. Masalah ini bisa teratasi karena ada cara lain input data tersebut yaitu secara manual, input langsung ke webnya tapi ada kekurangan jika tidak terkoneksi dengan internet tidak bisa. Satu bulan sudah bekerja secara manual, terdengar kabar akan ada perombakan file spreadsheet dan adanya bintek file tersebut ke kecamatan-kecamatan, hari H pun terjadi file spreadsheet benar-benar terombak dan lebih simple tapi file tersebut menggunakan format xls. Apakah akan terjadi hal yang sama? mencoba install WPS Office sesuai saran seorang teman di Facebook Meski sedikit gangguan saat install WPS Office, tapi terinstall juga.

Insiden, ikhlas dan i think diffrent

WPS Office sudah terinstall, saatnya untuk uji coba. Pertama mencoba mengisi file spreadsheet tersebut di LibreOffice, setelah selesai dan mencoba upload filenya dan masih sama The filetype you are attempting to upload is not allowed, saya kira setelah ada perombakan hal ini tidak terjadi lagi tapi entahlah. Masih ada senjata lain dengan mencoba WPS Office, namun kali ini bukan peringatan tipe file yang tidak diperbolehkan, tapi sebuah status error dari kode dari aplikasi tersebut yang pada intinya ada sebuah controller pembaca file spreadsheet tidak dapat membaca tipe file dari WPS Office yang berformat xls. Rasa kesal tak bisa terelak lagi ketika mendapat tugas input 3 file data dawis spreadsheet, membuat saya menjadi lebih lamban dalam bekerja. Sebenarnya data ada 2 bentuk satu masih belum digital berupa buku dan dan dan sudah digital berupa file spreadsheet yang harus di pindah ke spreadsheet dari developer baru di upload. Untuk yang berupa buku bisa diinput langsung atau melalui spreadsheet. Sebenarnya saya ingin berkonsultasi dan ingin menanyakan jalan keluar masalah ini ke developernya, tapi apa daya saya hanya minoritas, menurut saya mungkin saja akan menyarankan memakai Microsoft Ofiice dan Windows.
