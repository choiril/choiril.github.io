Variable Dan Printing

February 2, 2018

Reading time ~2 minutes

Sekarang belajar variabel dan print lanjutan, untuk hari ke-5 ini beajar tentang format string. Sebelum lebih lanjut lagi, kita akan berkenalan dengan string, untuk mengenali string cukup mudah yaitu adanya tanda “ (petik dua) disetiap awal kata. Contoh: “Ini adalah string”. String dapat kita kirim ke web server, simpan string ke dalam file dan masih banyak lagi.

Sedangkan untuk format string berguna untuk memanggil variable didalam string, pemnaggilan variablenya cukup mudah dengan penambahan tanda {}, misal {nama_variabel}. Dan ditambah huruf f kecil sebelum tanda “. Contoh penulisan: _f”ini adalah string {namavariabel}”. Untuk lebih jelasnya langsung dilihat pada kode dibawah, dimana kodenya sudah saya beri komentar dan sudah sesuai dengan study drill: 

#variable dan nilai
#nama variable haruslah huruf tidak boleh angka dan tidak ada spasi
nama = 'Choiril'
umur = 23
berat = 50 #kilogram
tinggi = 162 #centimeter
mata = 'hitam'
rambut = 'hitam'
kulit = 'sawo matang'

#penggunaan format string
#peletakkan format berada didepan string
#f adalah simbol dari format
print(f"Nama saya {nama}.")
print(f"Tinggi badan {tinggi}.")
print(f"Berat badan {berat}.")
print(f"Umur sekarang adalah {umur} tahun.")

#pengguanaan operator matematika dan format string
total = umur + berat + tinggi
print(f"Ketika umur, berat badan dan tinggi badan dijumlahkan akan menghasilkan {total}.")
#konversi ukuran
#disini akan merubah berat badan ke gram dan tinggi badan ke meter
konversi_berat = berat * 1000
konversi_tinggi = tinggi / 100
#penggunaan format string sesuai studi drill
print(f"Berat badan jika dikonversi ke gram adalah {konversi_berat} gram.")
print(f"Tinggi badan jika dikonversi ke meter adalah {konversi_tinggi} meter")

Pada kode diatas ada konversi ukuran, konversi ukuran adalah bagian dari study drill yang mana terdapat 2 pertanyaan untuk tidak menggunakan tanda underscore _ pada nama variabel selain konversi.
