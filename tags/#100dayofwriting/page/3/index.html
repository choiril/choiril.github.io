Variable Dan Printing

Sekarang belajar variabel dan print lanjutan, untuk hari ke-5 ini beajar tentang format string. Sebelum lebih lanjut lagi, kita akan berkenalan dengan string, untuk mengenali string cukup mudah yaitu adanya tanda “ (petik dua) disetiap awal kata. Contoh: “Ini adalah string”. String dapat kita kirim ke web server, simpan string ke dalam file dan masih banyak lagi. Sedangkan untuk format string berguna untuk memanggil variable didalam string, pemnaggilan variablenya cukup mudah dengan penambahan tanda {}, misal {nama_variabel}.
