Else if Pada Python

Hari ke-26 ini masih tentang if statement tetapi ada penambahan sintaks yaitu else-if, else-if ini berguna ketika terdapat banyak pilihan atau percabangan. Misal kita terdapat 4 pilihan, else-if ini yang akan digunakan. Else-if ini akan mnegecek satu persatu kodenya jika kode benar langsung dicetak outputnya, jika pilihan pertama masih belum benar maka akan lanjut ke pilihan kedua, dan begitu seterusnya sampai ada nilainya menghasilkan True (benar). Karena materi ini cukup mudah jadi artikelnya tidak terlalu panjang dan langsung ke hasil studynya.

If Statement Pada Pyhton

Hari ke-25 ini belajar tentang if-statement, atau sering disebut dengan bentuk kondisional. If-statement berlaku jika ada sebuah percabangan atau pilihan, misal apel ada lima buah dan anggur ada 10 buah. Jika apel lebih banyak dari banyak anggur maka yang dimakan lebih dahulu anggur, kurang lebih seperti itu penggunaan if-statement. Tak hanya Python saja, bahasa pemrograman lain juga juga menggunakan if-statement saat menghadapi sebuah pilihan ata percabangan. Jawaban dari study drill :
