Penggunaan Github dan Git Pemula

Sebuah cerita nyata mengenal dan menggunakan Github dan Git dari seorang pemula. Awal penasaran tentang teknologi git yang begitu menggoda para developer dan cara instalasi git pada PCLOS. Berkenalan dengan Github Apa itu Github?Github sebenarnya hampir mirip dengan sosial media hanya saja berisi banyak repositori sumberkode yang dikerjakan dan atau ditulis bersama untuk pengembangan perangkat lunak yang menggunakan sistem pengontrol versi git sehingga juga disebut social coding. Di github ini kita bisa punya pengikut dan mengikuti orang,memberikan umpan, wiki, grafik pengembangan, dan berkonstribusi dalam pembuatan sebuah perangkat lunak selain itu pula kita juga bisa membuat web statis dengan github pages.
