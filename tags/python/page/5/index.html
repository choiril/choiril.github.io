Bermain ASCII Python

Hari kesepuluh ini cukup lama belajarnya karena bermain tentang ASCII, menurut Wikipedia ASCII singkatan dari American Standart Code for Interchange atau Kode Standar Amerika untuk Pertukuran Informasi, ASCII dibaca dengan ejaan(‘aeski/ ass-kee’). ASCII ini merupakan bentuk huruf dan simbol dalam bentuk bilangan biner 7 bit. Jadi dengan ASCII kita bisa bermain kode rahasia, walau cukup mudah bagi mereka yang sudah terbiasa dengan ASCII. Tapi bagi saya ASCII ini adalah hal baru yang asyik yang membuatnya saya sedikit bingung karena kode ASCII yang saya dapat saat di print tidak sesuai dengan haeapan, entah kenapa.
